# SKILLS.md for spring-boot-playground

## AI Agent Skills and Capabilities

This document outlines the skills and capabilities of the AI coding assistant for this Spring Boot project.

### Core Skills
- **Spring Boot Development**: Expert knowledge of Spring Boot framework, including configuration, auto-configuration, and starters
- **Java Programming**: Proficient in Java 17+ with modern features, including records, sealed classes, and pattern matching
- **Maven Build System**: Full understanding of Maven lifecycle, dependencies, plugins, and multi-module projects
- **REST API Development**: Skilled in creating RESTful APIs using Spring Web MVC, with proper HTTP methods, status codes, and content negotiation
- **Database Integration**: Experience with Spring Data JPA, Hibernate, and various database configurations (H2, PostgreSQL, MySQL)
- **Testing**: Comprehensive knowledge of unit testing with JUnit 5, integration testing with Spring Boot Test, and mocking with Mockito
- **Security**: Understanding of Spring Security for authentication, authorization, and common security patterns
- **Microservices**: Familiar with Spring Cloud for service discovery, configuration management, and circuit breakers
- **Containerization**: Knowledge of Docker and Kubernetes for containerizing Spring Boot applications
- **Monitoring**: Expertise in Spring Boot Actuator for health checks, metrics, and monitoring endpoints
- **Performance**: Skills in optimizing Spring Boot applications, including caching, connection pooling, and profiling

### Specialized Skills
- **Rate Limiting**: Implementation of rate limiting using filters, buckets, or external services
- **Caching**: Integration with Redis, Ehcache, or Caffeine for application caching
- **Messaging**: Experience with RabbitMQ, Kafka, and Spring Integration for asynchronous messaging
- **Reactive Programming**: Proficiency with Spring WebFlux and Project Reactor for reactive applications
- **Cloud Deployment**: Knowledge of AWS, Azure, or GCP deployment strategies for Spring Boot apps
- **API Documentation**: Generation of OpenAPI/Swagger documentation with SpringDoc
- **Logging**: Configuration of structured logging with Logback and integration with ELK stack
- **Configuration Management**: Use of Spring Cloud Config, Consul, or Kubernetes ConfigMaps
- **Circuit Breaker**: Implementation of resilience patterns with Resilience4j or Hystrix

### Development Practices
- **Clean Code**: Writing maintainable, readable, and well-documented code
- **Design Patterns**: Application of appropriate design patterns (Factory, Builder, Strategy, etc.)
- **SOLID Principles**: Adherence to Single Responsibility, Open-Closed, Liskov Substitution, Interface Segregation, and Dependency Inversion
- **TDD/BDD**: Test-Driven Development and Behavior-Driven Development practices
- **Code Reviews**: Understanding of code review best practices and common anti-patterns
- **Refactoring**: Skills in improving code structure without changing functionality
- **Version Control**: Proficient use of Git with branching strategies and pull request workflows

### Tools and Technologies
- **IDEs**: IntelliJ IDEA, Eclipse, VS Code with Spring Boot extensions
- **Build Tools**: Maven, Gradle
- **Databases**: H2, PostgreSQL, MySQL, MongoDB
- **Message Brokers**: RabbitMQ, Apache Kafka
- **Cache**: Redis, Ehcache
- **Monitoring**: Prometheus, Grafana, ELK Stack
- **Containerization**: Docker, Docker Compose, Kubernetes
- **CI/CD**: Jenkins, GitHub Actions, GitLab CI
- **Cloud Platforms**: AWS (EC2, ECS, Lambda), Azure, Google Cloud

### Communication Skills
- **Technical Writing**: Clear documentation, README files, and API specifications
- **Code Comments**: Meaningful comments explaining complex logic
- **Commit Messages**: Descriptive commit messages following conventional commits
- **Issue Tracking**: Use of GitHub Issues, Jira, or similar for task management
- **Collaboration**: Working effectively in team environments with code reviews and pair programming

### Learning and Adaptation
- **Continuous Learning**: Staying updated with latest Spring Boot releases and Java features
- **Problem Solving**: Analytical thinking and debugging complex issues
- **Research**: Ability to research and implement new technologies and frameworks
- **Adaptation**: Quickly adapting to new project requirements and technologies
